# Technology, Media & Telecom (TMT)

This vertical covers agentic workflows in the **NAICS 51 (Information)** sector, including software, cloud/infrastructure, media/content platforms, and telecommunications.

Use this folder to find workflows that are common in high-scale digital services:

- incident response and reliability
- case/ticket workflows
- trust & safety operations
- platform governance for tool-using agents

## Sub-verticals

> Use cases that don't fit a specific sub-vertical go in `general/`.

## Contributing here

- Put new use cases under the most relevant sub-vertical.
- Each use case must have a **SAFE Use Case ID** and **NAICS 2022 code(s)** in its metadata.
- Update the canonical registry: `use-cases.naics2022.crosswalk.json`.

See the root `README.md` and `CONTRIBUTING.md` for the required structure and safety rules.
